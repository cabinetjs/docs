### Overview [#overview]

You can specify actual SNS and web services to be crawled in the `dataSource` field. The field value must be in array
format, and each item must have the following common format.

| Name   | Description                              |
|--------|------------------------------------------|
| `type` | Given type of each data source           |
| `name` | Name of each data source _(for logging)_ |

<Callout type="warning">
    `name` field of each data source item must be unique.
</Callout>

### Example [#example]

```json5
{
    // ...
    dataSource: [
        {
            "type": "example",
            "name": "test-name"
        },
        {
            "type": "example",
            "name": "test-name2"
        }
    ],
    // ...
}
```
