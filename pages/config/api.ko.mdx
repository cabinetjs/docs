# `api` 필드 [#index]

| 항목     |   설명   |
|--------|--------|
| 데이터 타입 | Object |
| 필수     |   ✅    |

### 개요 [#overview]

`api` 필드는 CabinetJS의 API 서버를 구성하는데 사용됩니다. API 서버는 CabinetJS 애플리케이션과 상호작용할 수 있는 외부 클라이언트
애플리케이션을 개발하는데 사용될 수 있습니다.

### 예제 [#examples]

```json
{
    "api": {
        "type": "graphql",
        "port": 3000,
        "endpoint": "/graphql",
        "playground": true
    }
}
```
