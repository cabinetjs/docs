### Overview [#overview]

`api` Field is used to configure the API server of the CabinetJS application. The API Server is can be used to develop
external client applications that can interact with the CabinetJS application.

### Examples [#examples]

```json
{
    "api": {
        "type": "graphql",
        "port": 3000,
        "endpoint": "/graphql",
        "playground": true
    }
}
```
