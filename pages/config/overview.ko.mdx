# 설정 파일 [#configuration]

### 설명 [#description]

CabinetJS는 설정 파일을 통하여 이용 경험을 개인화 할 수 있습니다. 설정 파일을 통해 아카이브 할 SNS 및 웹 서비스, 크롤링 주기, 아카이브 할
컨텐츠 필터링, 웹 API 서버 등을 설정할 수 있습니다.

### 설정 파일 경로 [#path]

CabinetJS를 실행하게 되면, 현재 디렉토리에 있는 `cabinet.config.json` 파일을 찾게 됩니다. `-c`, `--config` 옵션을 통해 설정 파일의
경로를 지정할 수 있습니다.

```bash
$ cabinet -c /path/to/config.json
```

만약 해당 경로에 설정 파일이 존재하지 않는다면, CabinetJS는 [새로운 설정 파일을 생성](#default-settings)하게 됩니다.

### 기본 설정 값 [#default-settings]

```json
{
    "dataSources": [],
    "crawlInterval": 1800000,
    "storage": {
        "type": "local",
        "path": "./downloads"
    }
}
```
