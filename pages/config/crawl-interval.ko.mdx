# `crawlInterval` 필드

| 항목        |        설명        |
| ----------- | :----------------: |
| 데이터 타입 | Int, String |
| 필수        |         ✅         |

### 개요

`crawlInterval` 필드는 크롤링 작업이 어느 주기로 실행 될지 명시할 수 있습니다. 필드 값은 숫자 값이나 문자열 값의 형식을 지켜야 합니다.

만약 해당 필드가 정수형 값으로 설정 되었을 경우 크롤링 주기는 밀리초 단위로 설정 되며, 문자열 값으로 설정 되었을 경우 크롤링 주기는
`vercel/ms` [지원 단위](https://github.com/vercel/ms)로 설정 하거나 Cron 표현식을 사용하여 설정 할 수 있습니다.

CabinetJS는 `crawlInterval` 필드에 지정된 값을 기준으로 크롤링 주기를 설정하며, 크롤링 주기가 돌아올 때 마다 크롤링을 시작합니다.
만약 크롤링이 이미 진행 중 일 경우 다음 크롤링 주기가 돌아올 때 까지 크롤링을 시작하지 않습니다.

### 유효한 값

`crawlInterval` 필드의 값은 다음과 같은 형식을 지켜야 합니다.

- 정수형 값
- `ms`, `s`, `m`, `h` 단위로 표현된 문자열 값 [(참고)](https://github.com/vercel/ms)
- Cron 표현식

### 예시

```json
{
    "crawlInterval": 60000
}
```

위 설정은 크롤링 주기를 1분으로 설정합니다.

```json
{
    "crawlInterval": "12m"
}
```

위 설정은 크롤링 주기를 12분으로 설정합니다.

```json
{
    "crawlInterval": "0 */5 * * * *"
}
```

위 설정은 매 5분마다 크롤링을 시작합니다.
