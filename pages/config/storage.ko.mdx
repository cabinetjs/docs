# `storage` 필드

| 항목     |   설명   |
|--------|:------:|
| 데이터 타입 | Object |
| 필수     |   ✅    |

### 개요

`storage` 필드는 크롤링 한 데이터 중 이미지 및 동영상과 같은 데이터를 저장할 위치를 지정할 수 있습니다.

### 예시

```json
{
    "storage": {
        "type": "local",
        "path": "/path/to/storage"
    }
}
```
