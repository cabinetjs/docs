# 로컬 저장소

### 개요

로컬 저장소는 로컬 파일 시스템에 저장되는 저장소입니다.

### 설정 데이터 구조

설정 데이터는 다음과 같은 구조를 가집니다.

```json
{
  "type": "local",
  "path": "/path/to/local/storage"
}
```

#### `type` 필드 [#type-field]

| 항목     | 내용      |
|--------|---------|
| 데이터 형식 | "local" |
| 필수 여부  | ✅       |

`type` 필드는 저장소의 타입을 지정합니다. 이 값은 다른 저장소 종류와 구분하기 위해 사용되기 때문에, `local` 이외의 값은 사용할 수 없습니다.

#### `path` 필드 [#path-field]

| 항목     | 내용     |
|--------|--------|
| 데이터 형식 | String |
| 필수 여부  | ✅      |

`path` 필드는 저장소의 경로를 지정합니다. 이 경로는 로컬 파일 시스템의 경로를 사용합니다. 해당 경로가 존재하지 않을 경우, 최초 실행 시
자동으로 생성됩니다.

상대 경로 및 절대 경로를 모두 사용할 수 있습니다. 상대 경로의 경우, CabinetJS를 실행한 디렉토리를 기준으로 합니다.

### 예제

```json
{
  "type": "local",
  "path": "/path/to/local/storage"
}
```

위 예제는 `/path/to/local/storage` 경로에 로컬 저장소를 생성합니다.
