# 시작하기 [#getting-started]

### 준비 사항 [#prerequisite]

CabinetJS는 NodeJS 환경에서 동작합니다. 따라서 NodeJS가 먼저 설치되어 있어야 합니다. NodeJS가 설치되어 있지 않다면
[NodeJS 공식 홈페이지](https://nodejs.org/ko/)에서 설치할 수 있습니다. NodeJS의 최소 버전은 `18.16.0` 이상이어야 합니다.

### 설치 [#install]

NodeJS 패키지 관리자를 통해 CabinetJS를 설치할 수 있습니다. `npm`, `yarn`, `pnpm` 등을 사용할 수 있습니다.

<Tabs items={["npm", "yarn", "pnpm"]}>
    <Tab>
        ```bash
        npm install -g @cabinet/cabinet
        ```
    </Tab>
    <Tab>
        ```bash
        yarn global add @cabinet/cabinet
        ```
    </Tab>
    <Tab>
        ```bash
        pnpm install -g @cabinet/cabinet
        ```
    </Tab>
</Tabs>

위 명령어를 통해 CabinetJS를 설치하면, 안정적인 버전의 CabinetJS가 설치됩니다. 만약 최신 개발 버전의 CabinetJS를 설치하고 싶다면,
`@dev` 태그를 명령어 뒤에 붙여주면 됩니다.

<Callout type="warning">
    보통의 경우 안정적인 버전의 CabinetJS를 설치하는 것을 권장합니다. 최신 개발 버전은 안정적이지 않을 수 있습니다.
</Callout>

<Tabs items={["npm", "yarn", "pnpm"]}>
    <Tab>
        ```bash
        npm install -g @cabinet/cabinet@dev
        ```
    </Tab>
    <Tab>
        ```bash
        yarn global add @cabinet/cabinet@dev
        ```
    </Tab>
    <Tab>
        ```bash
        pnpm install -g @cabinet/cabinet@dev
        ```
    </Tab>
</Tabs>

### 실행 [#run]

CabinetJS를 성공적으로 설치했다면, `cabinet` 명령어를 통해 CabinetJS를 실행할 수 있습니다. CabinetJS가 실행되면, 설정 파일을 현재
디렉토리에서 찾아서 불러오게 됩니다. 지정된 설정 파일이 없다면, 기본 값을 포함한 새로운 설정 파일을 생성하게 됩니다.

설정 파일에 대한 자세한 내용은 [설정 파일](/ko/config/overview)을 참고하세요.

```bash
$ cabinet
```

`--help` 명령줄 옵션을 통해 CabinetJS의 CLI 명령줄 도움말을 확인할 수 있습니다.

```bash
$ cabinet -h

Usage: cabinet [options]

Options:
  -c, --config <path>  Path to config file (default: "./cabinet.config.json")
  -h, --help           display help for command
```
